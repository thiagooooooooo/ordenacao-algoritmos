# Trabalho Prático – Algoritmos de Ordenação

Este repositório contém a implementação em C dos algoritmos **Bubble Sort** e **Selection Sort**, desenvolvida como parte de um exercício prático da disciplina.

## Objetivo
Implementar e comparar os algoritmos de ordenação Bubble Sort e Selection Sort, analisando o desempenho em vetores de diferentes tamanhos.

Os testes foram realizados com vetores preenchidos em ordem decrescente (pior caso), considerando os seguintes tamanhos:
- 100
- 1000
- 10000./ordenacao


Para cada execução, foram registrados:
- Tempo de execução (em milissegundos)
- Número de comparações
- Número de movimentações

## Tecnologias Utilizadas
- Linguagem C
- Biblioteca padrão `<time.h>` para medição do tempo de execução
- Controle de versão com Git
- Repositório hospedado no GitHub
